## 휴게소 음식 간편주문 서비스
  - 2020.08.11 ~ 2021.08.27 (2주)

<br>

## 💁‍♀️ 서비스 소개
  - 휴게소 음식 주문으로 인한 기다림을 해소하기 위해 만든 미리 주문 서비스
  - 지도에서 휴게소 위치 확인
  - 내 위치에서 가까운 휴게소 4곳 확인
  - 휴게소 음식 주문과 간편 결제 + 장바구니

<br>

## 💻 기술 스택
> ### Front : html, css, javascript, jquery, ejs
> ### Back : node.js
> ### DB : mysql
